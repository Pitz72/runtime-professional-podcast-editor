# v0.5.1 — "Il Gran Consolidamento (Hotfix 1)"
**Data di rilascio:** 24 Febbraio 2026

---

## 🎯 Obiettivo
Questo hotfix risolve immediatamente due difetti critici di User Experience identificati subito dopo il rilascio della v0.5.0 nell'interazione con il File Bin.

---

## 🔴 Fix Critici e Miglioramenti

### 1. Drag & Drop dal File Bin bloccato/lento
**Problema:** Trascinare un file audio dal File Bin nella Timeline falliva in silenzio o causava pesanti instabilità (blocchi) del processo di Drag and Drop di React/Chromium.
**Causa:** Il componente `FileBin.tsx` passava l'intero oggetto `AudioFile` serializzato via `JSON.stringify` al `dataTransfer`. Poiché l'oggetto conteneva tra le sue proprietà interne un `AudioBuffer` completo (i dati d'onda binari grezzi caricati in RAM), il browser tentava di serializzare gigabyte di buffer audio in formato JSON testuale, risultando in un drop fallito e un blocco architetturale. L'editor si aspetta solo una referenza per ricaricarselo, non l'intera clona del buffer.
**Soluzione:** La serializzazione per il trascinamento ora codifica esclusivamente l'ID del file (`{ id: file.id }`). L'operazione torna istantanea e permette il corretto linkaggio nella Timeline.

### 2. File Bin non cliccabile per visualizzare Proprietà
**Problema:** Cliccare su un file nel File Bin non mostrava alcuna informazione nel rannello delle proprietà (`PropertiesPanel`), rimanendo bloccato sulla schermata "Select a track or a clip".
**Causa:** Il pannello `PropertiesPanel.tsx` e la state dell'editor (`Editor.tsx`) permettevano la ricezione esclusivamente di identità di tipo `track` o `clip`. I file originali presenti nel bin non erano un tipo supportato per la selezione.
**Soluzione:**
1. Aggiornato lo stato globale di `selectedItem` in UI per accettare anche il tipo `file`.
2. Aggiunto l'evento `onClick` sugli elementi `<li>` del `FileBin` esportato via props.
3. Creata la vista `renderFileProperties` in `PropertiesPanel` per mostrare le caratteristiche natie del file (Nome, Formato Originario/Type, Durata in Secondi esatta).
